# Next.js Commands

Create T3 stack Next.js projects with customizable options.

## Create Project

```bash
gerd next create <project-name>
```

Scaffolds a [T3 Stack](https://create.t3.gg) project with interactive configuration.

### Configuration Options

You'll be prompted to select:

| Option | Choices | Default |
|--------|---------|---------|
| **Router** | App Router, Pages Router | App Router |
| **API** | tRPC, None | tRPC |
| **ORM** | Drizzle, Prisma, None | Drizzle |
| **Database** | postgres, mysql, sqlite, planetscale | postgres |
| **Auth** | NextAuth.js, None | None |
| **Styling** | Tailwind CSS, None | Tailwind CSS |

## Quick Start

```bash
# Create new project
gerd next create my-app

# Navigate and install
cd my-app
pnpm install

# Start development
pnpm dev
```
