# React Commands

Create React projects with your choice of bundler.

## Create Project

```bash
gerd react create <project-name>
```

### Bundler Options

| Bundler | Description |
|---------|-------------|
| **Vite** | Fast build tool with templates for React/TypeScript/SWC |
| **Bun** | All-in-one runtime with built-in React support |

### Vite Templates

| Template | Description |
|----------|-------------|
| `react` | JavaScript |
| `react-ts` | TypeScript (default) |
| `react-swc` | JavaScript + SWC |
| `react-swc-ts` | TypeScript + SWC |

### Package Managers

When using Vite: npm, pnpm, or bun

## Quick Start

```bash
# Create with Vite (default)
gerd react create my-app

# Follow prompts, then:
cd my-app
pnpm install
pnpm dev
```
